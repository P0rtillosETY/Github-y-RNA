Tarea 1: Git y documentación de RNA.
Para esta actividad, utilicé la base del código de Michael Nielsen (metwork.py) para entrenar una red neuronal que clasifica números del dataset.
Primero corrí el entrenamiento para verificar que todo estuviera en orden y luego me enfoqué en la parte de documentación y traducción.
Añadi explicaciones sobre las funciones  y use:
network.py: El código de la red neuronal y el aprendizaje (Backpropagation).
main.py: El archivo que ejecuto para iniciar el entrenamiento.
mnist_loader.py: Carga las imágenes para que la red las entienda.
mnist.pkl.gz: La base de datos con los números escritos a mano.
resultado1.png: Captura de pantalla del resultado final.
.gitignore: Evita que se suban archivos temporales innecesarios.
